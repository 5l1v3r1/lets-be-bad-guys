Malicious File Access

{% endblock %} {% block extra_head %} {% if msg %} {% endif %} {% endblock %} {% block content %}

Many sites use a dynamic url to retrieve files for the user. The most common implementation seems to be pulling user images like their profile picture. Instead of using an id or token these sites sometimes use partial file paths. A clever individual could spot this and try to get our DB passwords, /etc/passwd file or other sensitive data on our filesystem.

Simple exercise: get the settings.py file to display on the screen.
