Bypassing Access Control

An application might use forwards to route requests between different parts of the site. To facilitate this, some pages use a parameter to indicate where the request should be sent if a transaction is successful. This can allow an attack to craft a URL that will bypass the site's usual access controls and forward the attacker into a sensitive or restricted area of the site (like an admin app).

Our vulnerable codebase has this kind of flaw. Exploit it to access our super-secret 'admin' area.
